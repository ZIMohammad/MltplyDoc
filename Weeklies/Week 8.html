>> >> Rock Modeling: 1 Hour


>> >> >> Process

>> >> >> After asking the rest of the art team which rock modeling method yielded the best results, I took the method they liked and made several rock models with it.
>> >> >> The rock modeling method in question involed taking a cube, subdividing it multiple times, displacing its vertices procedurally using a voronoi mask, using a decimate modifier to simplify the mesh, using a remesh modeifier to even the polygon density, using a noise based displacement to add detail to the mesh, then finally using a decimate modifier to simplify the detailed mesh.
